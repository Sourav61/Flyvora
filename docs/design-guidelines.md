🔤 Typography System

Goal: Clarity + calm confidence

Font Style

Primary: Clean geometric sans (e.g., Inter / SF Pro style)

Tone: Neutral, readable, modern

Hierarchy

Element	Size	Weight	Usage
H1	32px	SemiBold	Page titles
H2	24px	Medium	Section headers
H3	18px	Medium	Sub-sections
Body	14–16px	Regular	Main content
Caption	12px	Regular	Meta info

Rules

Line height ≥ 1.5

Avoid dense text blocks

Use spacing to create hierarchy, not just size
